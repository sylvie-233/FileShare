# 更新日志

本项目的所有重要更改都将记录在此文件中。

## [0.1.0] - 2025-03-29

### 新增
- 📤 拖拽文件上传功能
- 📱 响应式设计，完美适配移动端
- 🔗 分享链接一键复制功能
- 📂 文件上传、下载、删除功能
- 🗑️ 一键清空所有文件功能
- 📊 显示文件大小和修改时间
- 🔔 操作成功/失败的实时通知
- 🎨 现代化渐变背景和流畅动画
- ⚡ 基于 FastAPI 和 Uvicorn 的高性能后端

### 技术栈
- FastAPI 0.115+
- Uvicorn
- Pydantic
- 原生 HTML/CSS/JavaScript
- uv 包管理

### API 端点
- `POST /api/v1/upload` - 上传文件
- `GET /api/v1/files` - 获取文件列表
- `GET /api/v1/download/{filename}` - 下载文件
- `DELETE /api/v1/files/{filename}` - 删除单个文件
- `DELETE /api/v1/files` - 清空所有文件
